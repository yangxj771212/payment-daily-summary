# 缴费日期会计汇总系统

上传 `.xls`、`.xlsx` 或 `.xlsm` 缴费明细，自动生成按日会计汇总 Excel。无需数据库。

## 固定规则

- 银行存款 = 每日实缴金额合计
- 预收报名费 = 业务类别为“预缴费”的每日实缴金额
- 预收报名费转收入 = 每日预缴金额合计
- 逐笔收入 =（该笔实缴金额 - 该笔预收报名费 + 该笔预缴金额）/ 1.03，四舍五入到分后再按日汇总
- 逐笔税 = 该笔收入 × 0.03，四舍五入到分后再按日汇总
- 导出的“缴费明细”工作表会列出每笔确认的收入和税，便于核对

输入表必须包含：`缴费日期`、`业务类别`、`预缴金额`、`实缴金额`。

## 部署到 Render

1. 把本目录所有文件上传到一个 GitHub 仓库根目录。
2. 登录 Render，选择 **New → Blueprint**。
3. 连接该 GitHub 仓库，确认创建 `payment-daily-summary` 服务。
4. 等待部署完成，打开 Render 提供的网址。

`render.yaml` 已包含构建、启动和健康检查配置，不需要数据库或环境变量。

## 本地测试

```bash
pip install -r requirements.txt
python app.py
```

浏览器打开 `http://127.0.0.1:5000`。
